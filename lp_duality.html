Transport Optimization

Minimize total shipping cost · Dual variables = marginal value of supply/demand capacity

Primal Problem
Minimize Z = Σ cᵢⱼ xᵢⱼ
Subject to:
  Σⱼ xᵢⱼ ≤ sᵢ (supply)
  Σᵢ xᵢⱼ ≥ dⱼ (demand)
  xᵢⱼ ≥ 0
Dual Problem
Maximize W = Σⱼ dⱼ vⱼ − Σᵢ sᵢ uᵢ
Subject to:
  vⱼ − uᵢ ≤ cᵢⱼ ∀i,j
  uᵢ ≥ 0, vⱼ ≥ 0

Healthcare Resource Allocation

Maximize patients treated · Dual variables = marginal value of adding one doctor or bed

Primal Problem
Maximize Z = Σⱼ xⱼ (patients)
Subject to:
  Σⱼ aᵢⱼ xⱼ ≤ bᵢ ∀i (resources)
  xⱼ ≥ 0
Dual Problem
Minimize W = Σᵢ bᵢ yᵢ
Subject to:
  Σᵢ aᵢⱼ yᵢ ≥ 1 ∀j
  yᵢ ≥ 0
yᵢ = shadow price of resource i

Energy Distribution

Minimize distribution cost · Dual = marginal cost of supplying extra MW at each node

Primal Problem
Minimize Z = Σ cᵢⱼ fᵢⱼ
Subject to:
  Σⱼ fᵢⱼ ≤ Gᵢ (generation cap)
  Σᵢ fᵢⱼ ≥ Dⱼ (demand met)
  fᵢⱼ ≥ 0
Dual Problem
Maximize W = Σⱼ Dⱼ λⱼ − Σᵢ Gᵢ μᵢ
Subject to:
  λⱼ − μᵢ ≤ cᵢⱼ
  λⱼ, μᵢ ≥ 0
λⱼ = locational marginal price (LMP)
